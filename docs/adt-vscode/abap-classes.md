# ABAP Classes and Interfaces

Learn about the features of ABAP classes and interfaces.

Context

In this documentation, you'll learn about the functionalities and features available within ABAP classes and interfaces.

Key Features

Occurrence Marker

	

The occurrence marker highlights occurrences of the same code element, like variables or methods, in your code. This is beneficial if you, for example, want to change all occurrences of an element at once. To do this, hover over one occurrence and open the context menu. Then, select Change All Occurrences or press Ctrl + F2 .




Formatter

	

Maintain the readability and quality of your code with the ABAP formatter. Customize your formatting rules in ABAP development tools for Eclipse and specify formatting options like syntax alignment, keyword casing (uppercase/lowercase), line length, and spacing. To format your code, open the context menu in your editor and choose Format Document or press Shift + Alt + F to format the document directly, or choose Format Document With... to select an alternative formatter.

To learn how to define formatting options in ABAP development tools for Eclipse, see Defining ABAP Formatting Options.




Outline View

	

The Outline view displays a hierarchical list of the internal structure of an ABAP class, interface, ABAP program, or other objects that are currently open in the ABAP source code editor. You can use the outline to navigate within a source-based development object or to get a quick overview of its structural elements.




Navigation

	

The navigation allows you to open the definiton of an element or subelement.

Place your cursor over an element and open the context menu. Choose Go to Definition. Alternatively, press F12.




Code Completion

	

Get in-line suggestions for your code. Just start typing and press Ctrl + Space once you'd like to display the list of suggestions next to your code. These suggestions could be variables or identifiers. Press Enter once you've selected an entry from the suggestions and the variable or identifier will appear at the same position.




Hover

	

Hover over an element to display element information such as subelements and short text.

To enable this feature, go to File  Preferences  Settings  Text Editor  Hover: Enabled and select on from the dropdown menu.
