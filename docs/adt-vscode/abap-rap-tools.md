# ABAP RAP Tools

The ABAP RESTful Application
Programming Model (in short RAP) defines the architecture for
efficient end-to-end development of SAP HANA-optimized OData
services (such as SAP Fiori apps)
in ABAP Cloud.
Unpublishing and Previewing Service Bindings


Realization in ABAP Development Tools for
Eclipse
Realization in ABAP Development Tools for Visual
Studio Code



A service binding is provided in a form-based editor.To perform a certain functionality, choose the relevant button,
for example Unpublish or
Preview....To publish a service, choose the Publish button.To unpublish a service, choose the Unpublish button.
A service binding is provided in a source-based editor.To perform a certain functionality, choose the relevant code lens
(for example, Unpublish or
Preview...) in the
source code.Follow then the subsequent steps:
To publish a service, choose the Publish code lens.
To unpublish a service, choose the Unpublish code
lens.

Previewing SAP Fiori Elements App


Realization in ABAP Development Tools for
Eclipse
Realization in ABAP Development Tools for Visual
Studio Code



To preview a SAP Fiori
Elements app, open a service binding. Select an Entity Set and Association and choose then the
Preview... button.The default browser is opened and displays the app.
To preview a SAP Fiori Elements
app, open a service binding and select and choose the Preview... code lens. The entity
dialog is opened.Select an entity. A dialog is opened which asks you where to open the application.Choose Open. The default browser is opened and
displays the app.
