# Activating Data Definitions

You activate a data definition to make it available for ABAP CDS
development.
Prerequisites
Ensure that the data definition is syntactically correct. The system performs a
syntax check before activation.

Context
You have created a new data definition and want to start developing your ABAP data
model.

ProcedureYou have the following options to activate data definitions:
Choose the Activate button from the
toolbar.

You can use the following keyboard shortcuts:


Ctrl
 + F3
 to activate one object

Ctrl
 + Shift
 + F3
 to activate all inactive objects


You can use the following commands in the Command Palette:

ABAP: Activate to activate one object
ABAP: Activate All to activate all inactive
objects


Results
The back-end system creates an active version of the CDS entity and stores it in the ABAP
Dictionary.
