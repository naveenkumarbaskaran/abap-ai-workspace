# Activating ABAP Development Objects

Use the activation tool to activate one or more inactive development objects.

Context
Note

Ensure that the development object you want to activate is syntactically correct. Before activation, the system performs a syntax check on the entire object. However, you can still activate development objects even if they contain syntax errors.

Procedure
You can activate a single development object in the following ways:

Select the Activate button.

Open the Command Palette with Ctrl + Shift + P , and then run the command ABAP: Activate.

Press Ctrl + F3 .

You can activate multiple development objects in the following ways:

Open the Command Palette with Ctrl + Shift + P , and then do the following:

Select ABAP: Activate All....

Select the development objects.

Select OK.

Press Ctrl + Shift + F3 , select the development objects, and then press Enter.
