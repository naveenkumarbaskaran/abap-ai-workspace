# Agentic AI for ABAP Development

Agentic AI for ABAP development focuses on building goal driven ABAP applications by using AI agents that can execute sequential tasks with minimal human intervention. These AI agents leverage external tools, contextual awareness, and adaptive decision-making capabilities to drive complex ABAP development workflows.

ADT MCP Server

The Model Context Protocol (MCP) is an open standard that enables AI applications to connect with external systems, tools, and data sources. For more information, see What is the Model Context Protocol (MCP)? .

The ADT MCP Server implements the MCP standard, offering MCP tools for ABAP development tasks such as creating ABAP development objects, creating transports, and generating complete RAP applications, thereby supporting AI agent-driven ABAP development. For more information, see Model Context Protocol Tools.
