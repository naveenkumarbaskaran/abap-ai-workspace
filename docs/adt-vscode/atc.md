# ABAP Test Cockpit

The ABAP Test Cockpit (ATC) is the main ABAP tool for quality assurance.
Using the ATC, you can check your ABAP development objects for many types of problems,
including syntax errors, potential problems in performance, sub-optimal or potentially
faulty programming, adherence to standards, errors in ABAP Unit testing, and many
others.
