# Getting Support from the Content Assist

In the CDS source code editor, the content assist proposes valid
keywords and identifiers that can be inserted into the source code at the cursor position.
Consequently, you can reduce the time spent on code editing and ensure that you are using
the valid syntax and source code elements.
The following content assist functions are provided for CDS objects:


Content Assist
Description



Code Completion
When you use code completion (
Ctrl
 + Space
) at a specific position in your ABAP source code, the code
completion popup will display:
All matching keywords
Semantic identifiers, such as the best matching data sources,
parameters, or elements of an ABAP Dictionary artifact
in the code completion popup.
