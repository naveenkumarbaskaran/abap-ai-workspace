# Displaying Code Element Information for Data Definitions

Use

The Hover widget displays details of CDS
entities and their elements used in the context of ABAP CDS.
You can use the Hover widget while programming
to get further details, for example, which element of a data source can be used for
creating a data model and how it is typed.
