# ABAP Core Data Services

ABAP Core Data Services (ABAP CDS) is a technology for defining and consuming semantically rich
and SAP HANA-optimized data models.
The subsequent content describes the usage of the tools to build such data models that are used to develop, for example, all
types of transactional SAP Fiori applications in context of the ABAP RESTful Application
Programming Model
(RAP).
