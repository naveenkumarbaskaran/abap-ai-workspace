# Checking Syntax in Data Definitions

The syntax check function checks whether the source code in data definitions is
syntactically correct or not.
Context
The syntax check function is automatically triggered when you start typing in the
CDS source code editor and enabled by default.

ProcedureTo trigger the syntax check, proceed as follows:
Open the CDS source code editor.

Start typing or editing source code.

Use the Activate button on the
toolbar.
Optional: 
To check the syntax of all open editors, choose the Activate
All button after this on the primary state bar.

Results
If errors are issued during the check, these will be issued to the Problems view. Besides, the code line with the error
is labeled with a decorator in the CDS source code editor.
