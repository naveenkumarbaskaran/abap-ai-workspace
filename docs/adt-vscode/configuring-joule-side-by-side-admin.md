# Configuring the SAP Joule for Developers, ABAP AI Capabilities for Side-by-Side Use (Administrators)

Learn more how you as an administrator can configure the SAP Joule for Developers, ABAP
AI capabilities for side-by-side
use and how you can support the developers in doing their necessary
configuration.
Prerequisites


You are an administrator with full access to SAP for Me.
You have purchased AI units in order to use SAP Joule for
Developers, ABAP AI capabilities enhanced by agentic AI
features (see SAP Note 3740750  ).


Procedure


Open SAP for Me and
navigate to Portfolio & Products.
Select the tab My Product Packages and choose
Technology Platform AI Agents and Features.
Select the tab Applications and choose the application SAP
Joule for Developers, ABAP AI capabilities.
Now you can Start Provisioning and enter the Solution
Details:
Select an Cloud Identity Services system.
Select your region.
Choose a Formation Name (the name is mandatory,
but has no further technical relevance). 

Activate the feature in SAP Central Business Configuration as described in the respective
documentation: Activating Business AI and Assigning Users. 
In the tab Resources you will find a new tenant providing its
Main URL (which will lead to the UI of the
application).
Provide that URL to the developers so they can configure SAP Joule for Developers, ABAP
AI capabilities for side-by-side
use in ABAP development tools for Visual
Studio Code (see Start).
