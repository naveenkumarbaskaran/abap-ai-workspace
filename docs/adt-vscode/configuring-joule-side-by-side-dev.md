# Configuring the SAP Joule for Developers, ABAP AI Capabilities for Side-by-Side Use

Learn more how you configure ABAP development tools for VS
Code (ADT) to use the
SAP Joule for Developers, ABAP
AI capabilities
side-by-side on base of a connection to the SAP Business Technology
Platform.
Prerequisites
Your SAP administrator has configured the subscription of the central BTP service on the
SAP Business Technology
Platform that provides
the latest SAP Joule for Developers, ABAP
AI capabilities (see Configuring the SAP Joule for Developers, ABAP AI Capabilities for Side-by-Side Use (Administrators)).
Your SAP administrator provides you an URL that refers to the SAP BTP. In case no URL is
specified, the SAP Joule for Developers, ABAP
AI capabilities run on-stack.


Context
As an ABAP developer, you want to use the latest SAP Joule for Developers, ABAP
AI capabilities for all destinations in
your ADT client.

Procedure
To enable the SAP Joule for Developers, ABAP
AI capabilities side-by-side, open the Settings. To do this, use the File  Preferences  Settings menu or use 
Ctrl
 + ,
.

Navigate to the Extensions  ABAP Development settings group.

Open the Adt  Joule  Url setting.

Enter the URL for SAP Joule for Developers, ABAP AI
capabilities to the subscription of the central BTP service on
the SAP Business Technology
Platform.

ResultsFor all destinations, you can now use the latest SAP Joule for Developers, ABAP
AI capabilities in the editor.
