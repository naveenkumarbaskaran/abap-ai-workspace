# Creating ABAP Interfaces

Learn how to create ABAP interfaces.

Context

Follow the steps below to create ABAP interfaces.

Procedure
To launch the creation dialog, open the Command Palette with Ctrl + Shift + P , choose ABAP: Create New ABAP Object..., then choose ABAP Interface. Alternatively, use the Ctrl + Shift + Alt + N and select the relevant entry from the list.
Specify Package Name, Name, and Description.
[Optional:] Specify an Interface.
Enter a transport request under Select Transport Request or create a new one.
Results
You have created an ABAP interface in the specified package.
