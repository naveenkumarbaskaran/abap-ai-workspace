# Editing ABAP CDS Source Code

Use

You define the different types of ABAP CDS source code for the definition of a CDS object in the
respective CDS source code editor.
In accordance to the type of CDS source code, you can use the subsequent features to develop
your data model.


Related Information


 Getting Support from the Content AssistIn the CDS source code editor, the content assist proposes valid
keywords and identifiers that can be inserted into the source code at the cursor position.
Consequently, you can reduce the time spent on code editing and ensure that you are using
the valid syntax and source code elements.Formatting Data DefinitionsYou format the source code of data definitions (for example, a CDS statement,
including the element list, Boolean expressions, JOINs,
association definitions, and so on) to structure the code and to improve its
readability.Checking Syntax in Data DefinitionsThe syntax check function checks whether the source code in data definitions is
syntactically correct or not.Displaying Code Element Information for Data Definitions
