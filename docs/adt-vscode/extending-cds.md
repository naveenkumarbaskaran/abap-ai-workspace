# Extending Data Models

You can extend CDS entities of the SAP standard in order to add customer-specific
functionality to a data model without resulting in modifications.
You can use the following CDS view extensions to extend CDS entities:

CDS Metadata Extensions (in
short: metadata extensions) to overwrite existing or to add new CDS annotations
to one or more elements or parameters of a CDS entity.
CDS Entity Extensions (in
short: entity extension) to extend an existing CDS entity using the respective
extend statement.
