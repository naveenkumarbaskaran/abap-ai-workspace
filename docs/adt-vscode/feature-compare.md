# Feature Compare

The following pages compare the features and use of ABAP development tools for VS
Code and
ABAP development tools for
Eclipse. The content is structured in tables which describe differences with
regard to user experience (UX), usability, technical realization, and so on.
Scope

The focus is on functionality and its integration for easy use and fast transition.
