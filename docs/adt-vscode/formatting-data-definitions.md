# Formatting Data Definitions

You format the source code of data definitions (for example, a CDS statement,
including the element list, Boolean expressions, JOINs,
association definitions, and so on) to structure the code and to improve its
readability.
Prerequisites



Use

The following use case(s) are supported for formatting source code:
Using the SAP standard profile: This predefined profile is
provided by default and can be used immediately.


Execution

To execute the data definition formatter, choose Format Document from the context menu at every position in your source code.
