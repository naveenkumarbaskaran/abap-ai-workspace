# Getting Started (Administrators)

Learn more how you can get SAP Joule for Developers, ABAP
AI capabilities and which business roles are
required.
Roles and Authorizations

To be able to use SAP Joule for Developers, ABAP
AI capabilities, the following prerequisites need to be met:
 Your system administrator must assign you to
a business role that contains the business catalog
SAP_A4C_BC_DEV_AIQ_PC.
Additionally, the latest version of ABAP development tools for
Eclipse is required.



SAP Joule for Developers,
ABAP AI Capabilities With and Without Agentic AI Features

Note that there are two different types of SAP Joule for Developers, ABAP
AI capabilities:
SAP Joule for Developers, ABAP AI capabilities on-stack in SAP BTP ABAP Environment,
SAP Joule for Developers, ABAP AI capabilities enhanced by agentic AI
features.
