# Installing ABAP Development Tools for VS Code

Learn how to install ABAP development tools for VS Code.

Prerequisites
Ensure the following technical requirements are met:

Component

	

Prerequisite




Operating System

	

Windows 10 or higher

Apple macOS 14 or higher




Microsoft VC++ Runtime

	

For Windows: Microsoft Visual C++ v14 Redistributable (x64)

Context
Install ABAP development tools for VS Code following these steps:
Procedure
Go to Download Visual Studio Code and choose your installer to download.
Install VS Code on your device.
To get the ABAP Development Tools for VS Code extension, navigate to the Extensions tab in the Activity Bar on the far left of the IDE.
In the search bar, search for ABAP Development Tools.
Install the extension and restart the IDE.
Results
You've installed ABAP development tools for VS Code. You can now continue with Establishing a System Connection via RFC Destination or Establishing a System Connection via HTTP Destination.
