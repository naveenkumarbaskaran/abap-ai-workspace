# Creating Number Range Objects

You can create and edit number range objects in the number range object editor.

Procedure
To launch the creation dialog, open the Command Palette with Ctrl + Shift + P and select ABAP: Create New ABAP Object. Alternatively, choose Ctrl + Shift + Alt + N and select the relevant entry from the list.
Select the Number Range Object (Type: NROB/NRO) entry from the list.
Enter or select the Package Name, the Name, and the Description.
Assign a transport request.
Results

In the selected package, the ABAP back-end system creates an inactive version of a number range object and stores it in the ABAP Repository.

In the Explorer, the new number range object is added to the Number Range Management node of the corresponding package node. As a result of this creation procedure, the source-based editor will be opened.
