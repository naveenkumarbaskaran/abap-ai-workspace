# Opening ABAP Development Objects

Open ABAP development objects in ABAP development tools for VS Code to view or edit them.

Procedure
Do one of the following to open an ABAP development object:

Open the Command Palette with Ctrl + Shift + P and select ABAP: Open Object....

Press Ctrl + Shift + A to directly open the quick pick dialog.

Enter the full object name or use object name patterns and apply filters to refine the result list. For more information, see Property Filter and Object Name Patterns.
Select the development object that you want to work with.
