# Recommendations and Constraints

This chapter gives a general overview of boundary conditions for SAP Joule for Developers, ABAP
AI capabilities.
General Constraints and Recommendations for Generative AI

Processes and features based on generative AI technologies are non-deterministic by
definition. Even when all measures are taken to ensure only expected and valid
results, the final result can't be guaranteed. For example, the same prompt can
generate different responses from a large language model at different points in
time, and hallucinations are possible with large language models. 


SAP Joule for Developers,
ABAP AI Capabilities

Capabilities and skills in SAP Joule for Developers, ABAP
AI capabilities
rely on different large language models as part of their implementation. The used
large language models can change at any point in time, which can lead to varying
results using the same input parameters or prompts.
