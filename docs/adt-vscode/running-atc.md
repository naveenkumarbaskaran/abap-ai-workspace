# Running ATC Checks

Learn how to run ATC checks in ABAP development tools for VS
Code.
Context
You can run ATC quality checks on the class or other development object that you are editing in the editor.

Procedure
Open the relevant development object in the editor.

Use the ABAP: Run ABAP Test Cockpit command in the
Command Palette. 




ResultsAfter completion of the ATC check run, the ATC findings are displayed in the
Problems view.
