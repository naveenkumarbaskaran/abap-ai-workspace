# Collecting Support Information

Learn how to collect disgnostic information.
Context
With the ABAP: Create Support File command you can collect diagnostic
data that help you and support teams gather relevant diagnostic and configuration
information when troubleshooting issues.

Procedure
In the Command Palette, select the ABAP:
Create Support File command.

A selection dialog appears offering to select the data in the support file.
Mandatory data is always included. For optional data, check or uncheck items as
needed.

Choose where to save the generated ZIP file.

Once saved, a notification appears allowing you to open the file location or
copy the file path.

Share the file with SAP support, if needed.

ResultsYou created a ZIP file containing diagnostic and configuration information that you can
share with support teams.
