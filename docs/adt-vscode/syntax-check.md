# Checking ABAP Syntax

The ABAP syntax check enables you to verify code-based development objects before activating them.

Syntax errors appear in the Problems view as ABAP Problems, while the corresponding code lines are highlighted with decorators in the ABAP object editor.

By default, the ABAP object editor automatically checks the syntax of a development object you are currently working on.

You can run manual syntax checks by opening the Command Palette with Ctrl + Shift + P and selecting ABAP: Check Object.
