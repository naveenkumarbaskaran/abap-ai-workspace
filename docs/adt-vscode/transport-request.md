# Adding ABAP Development Object to a Transport Request

You add a ABAP development object to a transport request to transfer it to another ABAP system. Then, in the target system the same object is used.

Context

You want to create a new or to edit an existing development object. For this, you create a new or assign an existing transport request for/to it.

Procedure
To create a new or assign an existing transport request, after creating an object, choose the package that has transport enabled.
You can choose one of following to choose the transport request:

Choose the transport request from the Select Transport Request list.

Choose a transport request from a different user by choosing the Select User icon.

Enter a request number to add your ABAP object to an existing request that is already created.

Create a new transport request and assign the created transport request to the object.

To modify an object, edit the object and save it.
Assign a transport.
To delete a development objects, open the context menu and select Delete Permanently and assign a transport.
